# Agent Guardrails

This is a sanitized copy of the `CLAUDE.md` from the frontend package of
[AirbaseHQ](https://airbasehq.com), a production AI hiring and interviewing
platform I spent seven months building.

The package is the recruiter-facing frontend. It is built with React, Vite,
TanStack Router, and TanStack Query.

**[Read the Frontend CLAUDE.md](frontend/CLAUDE.md)**

Internal names and private implementation details have been removed. The rules
used by coding agents in the real package remain.

Created by [Shahbaz Manzoor](https://shahbazmanzoor.com) from conventions refined
on real projects.
